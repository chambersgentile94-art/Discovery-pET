# Flujos principales - Discovery-pET

## Flujo 1: registrar usuario

1. El usuario abre la app.
2. Ingresa email y contraseña.
3. Confirma el registro.
4. Completa nombre y ciudad.
5. Accede al mapa principal.

## Flujo 2: publicar mascota perdida

1. El usuario presiona Crear reporte.
2. Selecciona Mascota perdida.
3. Carga foto.
4. Completa tipo de animal, título y descripción.
5. Indica zona donde se perdió.
6. Define nivel de urgencia.
7. Publica el caso.
8. El caso aparece en el mapa.

## Flujo 3: publicar animal visto

1. El usuario presiona Crear reporte.
2. Selecciona Animal visto.
3. La app obtiene ubicación actual.
4. El usuario carga foto y descripción breve.
5. Publica el caso.
6. Otros usuarios pueden verlo en mapa.

## Flujo 4: actualizar caso

1. El usuario abre la ficha del reporte.
2. Agrega una actualización.
3. Cambia el estado si corresponde.
4. El historial queda registrado.

## Flujo 5: denunciar publicación

1. El usuario abre un reporte.
2. Presiona Reportar publicación.
3. Indica motivo.
4. La denuncia queda pendiente de revisión.
5. Un administrador decide si corresponde ocultar el reporte.
